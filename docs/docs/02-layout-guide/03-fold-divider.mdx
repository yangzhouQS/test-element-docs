## 参数

### 属性

| 属性        | 说明       | 类型 | 可选值 | 默认值 |
| :--------- |:--------| :-----| :-----| :-----|
| v-model	| 折叠状态，`true`代表折叠，`false`代表展开	| boolean	| 	|  |
| title | 分割线标题| string | |  	|  |
| disable-fold | 禁用折叠功能| boolean |  |  	| false |
| content-position | 分割线内容出现的位置| emun | 'left' / 'right' / 'center' |  	| left |
| default-expand | 默认展开（折叠功能需打开）| boolean | |  	| false |

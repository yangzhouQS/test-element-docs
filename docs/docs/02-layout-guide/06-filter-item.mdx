# 元素布局
::: tip
横向布局组件
:::

##  基础用法

分为label区域和内容区域，可以设置label宽度和文本




```
<template>
  <filter-item label="材料名称" :label-width="'80px'" >
    <el-input v-model="input"  placeholder="请输入内容"></el-input>
  </filter-item>
  <filter-item label="材料名称材料名称材料名称" :label-width="'80px'" >
    <el-input v-model="input"  placeholder="请输入内容"></el-input>
  </filter-item> 
</template>
<script lang="ts" setup>
  import {ref} from 'vue'
  import { Plus, Edit, Delete } from '@element-plus/icons-vue'
  const input = ref('')
</script>
```
##  描述

打开`description`属性，启用描述功能，可以在单据详情页中使用




```
<template>
  <el-row>
    <el-col v-for="i in 12" :xs="24" :sm="12" :md="8" :lg="6" :xl="4">
      <filter-item label="材料名称" :label-width="'80px'" description>
        文本内容文本内容文本内容
      </filter-item> 
    </el-col>
  </el-row>
</template>
<script lang="ts" setup>
  import {ref} from 'vue'
  import { Plus, Edit, Delete } from '@element-plus/icons-vue'
  const input = ref('')
</script>
```



## 参数

### 属性

| 属性        | 说明       | 类型 | 可选值 | 默认值 |
| :--------- |:--------| :-----| :-----| :-----|
| label	| label文本	| string	| 	|  |
| label-width	| label宽度	| string	| 	|  |
| label-position| label位置	| string	| left/right	| right |
| description| 启用信息描述模式	| boolean	|	| false |
| size| 控件大小	| string	|base/small/large| small |

### 插槽

| 名称        |  描述 |
| :--------- |:-----|
|  default | 内容 | 
|  label | 自定义label内容 |

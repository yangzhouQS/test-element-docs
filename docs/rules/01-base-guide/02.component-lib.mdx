## 组件

### ElementPlus 通用组件库

:::info
ElementPlus 官方组件库，跟随官方组件一起更新
[查看文档](https://element-plus.org/zh-CN/)
:::
### ElementPro 扩展组件库

:::info
基于 ElementPlus，扩展了一些 ElementPlus 中不满足业务需求的组件

[//]: # ([查看文档]&#40;./docs.html#/&#41;)
:::

#### 安装

```bash
$ pnpm install @cs/element-pro
```

#### 样式引入

```html
  <link rel="stylesheet" href="./lib/element-pro/theme/index.css" />
```
#### 组件引入

- `import` 引入方式

```javascript
import ElementPro from '@cs/element-pro';
import App from './App'

const app = Vue.createApp(App)

app.use(ElementPro)

app.mount('#app')


```

- 使用头部引入方式

```html
  <script src="./lib/element-pro/element-pro.iife.js"></script>
```


```javascript
// index.ts
import App from './App'

const app = Vue.createApp(App)

app.use(ElementPro)

app.mount('#app')

```

}

### 三方组件库

项目中如果要引入第三方组件，评估组件大小，如果组件大于会给包增加大于100K的体积时考虑在.html头部挂载的方式引入。
如果较小直接在页面中引用手动注册使用。

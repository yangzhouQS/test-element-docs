# class 公共类

::: tip
提供常用的公共类，减少样式的重复写入
:::

## 公共类

```css
/* 页面根节点 */
.main-page {
  height: 100%;
  width: 100%;
  padding: 0px;
  background: var(--el-bg-color-page);
}
/* 内边距-中 */
.padding-base {
  padding: var(--el-layout-gap-base)
}
/* 内边距-大 */
.padding-large {
  padding: var(--el-layout-gap-large)
}
/* 内边距-小 */
.padding-small {
  padding: var(--el-layout-gap-small)
}
/* 清除左侧内边距 */
.padding-clear-left {
  padding-left: 0px;
}
/* 清除右侧内边距 */
.padding-clear-right {
  padding-right: 0px;
}
/* 清除上边距 */
.padding-clear-top {
  padding-top: 0px;
}
/* 清除下边距 */
.padding-clear-bottom {
  padding-bottom: 0px;
}
is-loading /* 可以让图标动态旋转 */
el-icon--left /* 图标靠左，图标右边增加5px的外边距 */
el-icon--right /* 图标靠右，图标左边增加5px的外边距 */

/* 危险按钮偏移类 */
.button-margin-left {
  margin-left: 50px !important;
}
/* 清除边框-左 */
.border-clear-left {
  border-left: 0px;
}
/* 清除边框-右 */
.border-clear-right {
  border-right: 0px;
}
/* 清除边框-上 */
.border-clear-top {
  border-top: 0px;
}
/* 清除边框-下 */
.border-clear-bottom {
  border-bottom: 0px;
}
```
